# RuralCare - Remote Healthcare Prototype

RuralCare is a modern web application prototype designed for vitals monitoring in rural areas.

## Features
- **Vitals Monitoring**: Real-time simulation of Body Temp and Heart Rate.
- **Triage**: Automated emergency assessment based on vital thresholds.
- **Consultation**: Simulation of secure doctor-patient video relay.
- **Theme Support**: Premium Light and Dark modes.

## Tech Stack
- HTML5, CSS3 (Custom Design System), JavaScript (ES6+), FontAwesome 6.
